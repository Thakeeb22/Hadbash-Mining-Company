# HADBASH MINING COMPANY LIMITED - Website

## Overview

This is the official website for **HADBASH MINING COMPANY LIMITED**, a diversified mining and resource extraction company based in Kaduna, Nigeria. The website showcases the company's mission, vision, core values, business sectors, team members, project portfolio, and provides contact information for potential clients and partners.

## Project Description

Hadbash Mining Company Limited operates across multiple sectors including:
- **Mining** - Extraction and processing of minerals
- **Mineral Processing & Gemstones** - Processing and refinement of raw minerals
- **Petroleum Products & Energy** - Oil and energy sector operations
- **Agriculture & Agro-Allied Products** - Agricultural ventures and related businesses

The website is built as a modern, responsive single-page application that effectively communicates the company's brand identity and services.

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server-side setup required for basic functionality

### Installation

1. Clone or download the project files
2. Ensure all image assets are in the `img/` folder
3. Open `index.html` in a web browser

### Usage

- Navigate using the top navigation bar
- Click hamburger menu icon on mobile devices
- Use contact form to send inquiries
- Click on gallery images to view projects
- Follow social media links in the footer
